# Libs, tricks and tips

## Установка
```shell
composer config repositories.olegvashkevich-ltt git https://github.com/OlegVashkevich/ltt.git
composer require olegvashkevich/ltt:dev-master
```
## Использование
```shell
vendor/bin/ltt
```
